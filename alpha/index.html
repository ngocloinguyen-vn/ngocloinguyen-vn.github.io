Alpha Diversity in R: Getting Started

Date: July 01, 2025

This tutorial introduces the basic concepts of alpha diversity analysis using R, with examples commonly applied to microbial amplicon sequencing data (ASVs or OTUs). The same workflow can easily be adapted to any ecological community dataset where rows represent samples and columns represent taxa.

1. What is Alpha Diversity?

Alpha diversity describes the diversity within a single sample or site. It is commonly used to evaluate both:

  • Richness – the number of observed taxa (species, OTUs, or ASVs).
  • Evenness – how evenly individuals are distributed among taxa.

Together, these metrics provide an overview of community complexity within each sample.

2. Species Accumulation and Rarefaction Curves

Species Accumulation Curve

A species accumulation curve shows the cumulative number of taxa detected as additional samples are included. It is useful for evaluating sampling effort and determining whether most diversity has been captured.

Rarefaction Curve

Rarefaction estimates expected richness after subsampling reads to a common sequencing depth. This allows fair comparisons among samples with different numbers of sequencing reads.

3. Common Alpha Diversity Indices

  • Observed Richness – Number of observed taxa.
  • Shannon Diversity – Considers both richness and evenness.
  • Simpson Diversity – Gives greater weight to dominant taxa.

4. Should You Normalize the Data?

The answer depends on the diversity metric and the biological question.

No normalization is generally required

  • Presence/absence richness using functions such as specnumber().

Normalization is recommended when

  • Sequencing depth varies substantially among samples.
  • Comparing abundance-sensitive diversity indices.
  • Calculating Richness, Shannon, Simpson, or Evenness.

Richness is particularly sensitive to sequencing depth because deeper sequencing detects more rare taxa. Shannon and Simpson indices are generally less affected because they incorporate relative abundances, but normalization is still recommended for unbiased comparisons.

5. Common Normalization Methods

  • Rarefaction – Random subsampling to equal sequencing depth. The most common approach for alpha diversity.
  • CSS normalization – Suitable for compositional microbiome data.
  • TSS normalization – Converts counts to relative abundances.
  • DESeq2 normalization – Primarily used for differential abundance analysis.

In this tutorial, rarefaction is used prior to calculating Normalized Richness and Shannon Diversity to minimize biases caused by unequal sequencing depth.

6. Alpha Diversity Workflow in R

A typical alpha diversity analysis using the vegan package follows these steps:

  1. Prepare the community matrix (samples in rows and taxa in columns).
  2. Import sample metadata.
  3. Filter rare taxa and low-quality samples if necessary.
  4. Rarefy or normalize the sequencing data.
  5. Calculate alpha diversity indices (Richness, Shannon, Simpson).
  6. Visualize results using boxplots, violin plots, or scatter plots.
  7. Perform statistical tests (t-test, Wilcoxon, ANOVA, Kruskal-Wallis, etc.).
